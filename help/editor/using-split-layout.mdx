---
title: "Using Split Layout"
---

The Split Layout feature allows you to display multiple clips, images, or frames on the screen at the same time. This is especially useful for reaction videos, side-by-side comparisons, or adding multiple perspectives to your content.

### Steps to Use Split Layout

**Step 1:** Open the editor for the clip you want to edit.

![Edit Captions1 Jp](/images/edit-captions1.jpg)

**Step 2:** On the top ribbon, click the **Layout** button.

![Aspect Ratio9 Jp](/images/aspect-ratio9.jpg)

**Step 3:** From the layout options, choose **Portrait Split Layout**.

![Split Ratio 1 Jp](/images/split-ratio-1.jpg)

**Step 4:** Adjust the size and position of each section to create the perfect framing.

![Split Ratio2 Jp](/images/split-ratio2.jpg)

---

That’s it\! You’ve now created a Split Layout. This is a simple but powerful way to make your videos more dynamic and engaging.
